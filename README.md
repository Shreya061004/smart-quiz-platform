# 🎯 Smart Quiz Platform

A full-stack web application that enables users to participate in quizzes, evaluate their performance, and manage quiz-related data efficiently. This project demonstrates the implementation of modern web technologies with a structured client-server architecture.

---

## 🚀 Features

* 📝 Interactive quiz interface
* 📊 Automatic score calculation
* 🔐 Secure user authentication (JWT-based)
* 📡 RESTful API integration
* ⚡ Fast and responsive UI
* 📂 Modular project structure

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* HTML5, CSS3
* JavaScript (ES6+)

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB

### Tools & Technologies:

* Git & GitHub
* Postman (API Testing)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```id="3mn5hm"
git clone https://github.com/Shreya061004/smart-quiz-platform.git
cd smart-quiz-platform
```

---

### 2️⃣ Setup Backend

```id="rmd8s1"
cd server
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```id="sz1weo"
cd client
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder and add the following:

```id="8r3ozt"
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
